Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header includes header of a page/section
div division includes a division or a section
ol ordered list includes an ordered list
ul unordered list includes an unordered list
li list item includes an item for a list
blockquote block quotation show a formatted quotation
br break enter line break

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make bold text <strong>sample sentence</strong> sample sentence
a anchor link to another page <a href="https://google.com">Google.com</a> google.com
img image display images <img src="images/randomimage.jpg" alt = "image placeholder" width="80" height="80">
code code display code snippet <code>sample code</code> sample code
span span color text <span style="color:green">colored text</span> colored text

Document Tags

Tag Name Purpose
html html Defines root of html documents
head head Defines head portion of the document
body body Defines main content of an html page
title title Defines the title of an html page
meta meta used to give metadata to provide info about a document
